Data Center Infrastructure Design
Data Center: suatu fasilitas yang digunakan untuk menempatkan sistem komputer dan komponen terkaitnya agar semua sistem informasi berjalan dengan baik. 
Aspek: 1. Lokasi : berada di luar radius mitigasi bencana/gunung berapi (>15 km), cukup tersamarkan dari pengenalann public (tidak terekspose)
2. Sarana Penunjang: catu daya dengan minimum 2 sumber pembangkit, generator listrik cadangan, uninterruptible power system (UPS)
3. Keamanan: CCTV, access control, pengamanan selama 24 jam, automatic gas fire suppression system
Tiers data center 1-4

System Infrastructure Design
Ada 3 tiers: web > apk > database
1. Web server: static vs dynamic content, serves web pages, web 2.0 and SaaS
2. apk: business logic, data manipulation, 
3. database: data storage

Virtualization using Virtual Machine
1. Lower number of physical servers and reduce hardware maintenance
2. Consolidation strategy, increase space 
3. prevent impact upgrades or changes application
4. Easily duplicated
5. Multiple OS on single hardware
